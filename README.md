# 🤖 Basic Chatbot (Python)

This project is developed as part of the CodeAlpha Artificial Intelligence Internship.
🚀 Project Overview
The Basic Chatbot is a simple rule-based chatbot built using Python. It interacts with users by responding to predefined inputs such as greetings, questions, and farewells.

This chatbot demonstrates how conversational systems can be created using basic programming logic without advanced AI.

✨ Features
- 💬 Interactive text-based conversation
- 🧠 Rule-based responses using if-elif conditions
- 🔁 Continuous conversation loop
- ⚠️ Handles unknown inputs gracefully
- 🚪 Exit option to end chat

🛠️ Technologies Used
- Python
- Conditional statements (if-elif)
- Functions
- Loops
- Input/Output handling
